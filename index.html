Reinforcement Learning Demo with Grid World Game

Agent học cách di chuyển từ vị trí xanh lá đến mục tiêu vàng, tránh chướng ngại vật đỏ

Thuật toán Q-Learning

🧠 Hàm học chính (Dòng 108-116, game.js):
updateQValue(state, action, reward, nextState)
Q(s,a) ← Q(s,a) + α[r + γ max Q(s',a') - Q(s,a)]

Các thành phần:

  • • α (alpha): Learning Rate - tốc độ học (0.1)
  • • γ (gamma): Discount Factor - giá trị tương lai (0.9)
  • • ε (epsilon): Exploration vs Exploitation (0.1)
  • • r: Reward nhận được từ hành động
🎯 Quá trình học (Dòng 118-170, game.js):
  1. Chọn hành động (chooseAction - dòng 86):

    Epsilon-greedy: 10% khám phá ngẫu nhiên, 90% khai thác tốt nhất

  2. Thực hiện hành động (step - dòng 118):

    Di chuyển agent và tính reward

  3. Cập nhật Q-value (updateQValue - dòng 108):

    Áp dụng công thức Bellman để học

  4. Lặp lại:

    Tiếp tục cho đến khi đạt mục tiêu hoặc hết step

💡 Cách Agent học:

1. Ban đầu: Agent di chuyển ngẫu nhiên, Q-values = 0

2. Khám phá: Agent thử các hành động khác nhau, nhận reward (+100 goal, -100 obstacle, -1 step)

3. Học hỏi: Q-values được cập nhật theo công thức Bellman

4. Tối ưu: Dần dần Agent biết đường đi tốt nhất (policy)
